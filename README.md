# Nexter // Luxury Real Estate 🏠

Nexter is a modern, high-end landing page for a luxury real estate agency. This project was specifically designed to master **CSS Grid Layout**, creating a complex and highly responsive interface that adapts seamlessly to any screen size without relying on external frameworks.

##  Live Demo
**[View Project Live]** -> (https://nexter-five-psi.vercel.app/)

---

##  Key Features
* **Complex 8-Column Grid:** Built a sophisticated main layout using an 8-column system with named grid lines.
* **Responsive Gallery:** A beautiful, asymmetric image gallery created using `grid-row` and `grid-column` spanning.
* **Advanced Responsive Design:** Implemented a completely fluid layout using `grid-template-columns: repeat(auto-fit, minmax(..., 1fr))` to minimize the need for media queries.
* **Modern Components:** Features custom-built house cards, a features section, and a sidebar navigation layout.
* **SVG Integration:** Utilized SVGs for high-quality icons and brand assets to ensure crisp visuals on retina displays.

##  Technical Stack
* **HTML5:** Clean, semantic markup tailored for grid-heavy layouts.
* **SASS (SCSS):** Structured using the 7-1 pattern for modularity.
* **CSS Grid:** Mastered `grid-template-areas`, implicit vs. explicit grids, and alignment properties (`align-content`, `justify-items`).
* **BEM Methodology:** Maintained a strict Block-Element-Modifier naming system for scalability.

##  Technical Highlights
* **The "Holy Grail" of Layouts:** Successfully managed a layout that includes a sidebar, header, features, story, homes, gallery, and footer using a single main grid.
* **Fluid Typography:** Implemented font scaling techniques that work in harmony with the grid containers.
* **Architecture:** Used SASS mixins to handle responsive breakpoints efficiently across the grid structure.

##  Project Context
This project was built during the **"Advanced CSS and Sass"** course by Jonas Schmedtmann. It represents the pinnacle of modern CSS layout techniques, moving beyond Flexbox to solve two-dimensional design challenges.
